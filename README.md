## Project Overview
This project is a web-based platform designed to connect workspace owners with coworkers for renting office environments, meeting rooms, and desks. The application supports specific user roles with workflows for listing, searching, and managing workspace rentals.

## Phase 1 Implementation Details
For Phase 1, the **Backend logic** is handled via a local data layer using JavaScript objects and arrays in `js/data.js` to act as a local web service for storing and retrieving user, property, and workspace information via `localStorage`.
